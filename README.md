# 🚀 Mini CRM System
A simple Mini CRM application to manage leads and track customer interactions with a clean UI.

## 🛠️ Tech Stack
- React (Vite)
- Node.js + Express
- MongoDB

## ⚙️ Run Locally

### Backend
cd backend  
npm install  
npm start  

### Frontend
cd frontend  
npm install  
npm run dev  

## 📌 Features
- Add, edit, delete leads  
- Notes management  
- Dashboard view
